Description

A standard Odoo component (mail.thread widget) for displaying a chronological log of messages, system notifications, and attachments related to a specific record. It also provides an interface for users to add new comments and schedule activities.

Design System Alignment

This component uses the design system's neutral color palette for backgrounds and text, primary colors for interactive elements like links and buttons, and standard tokens for typography, spacing, and borders.

Responsive Specifications

Mobile Adaptations

  • On screens < 768px, the composer action buttons stack vertically to fit narrow viewports.
  • The message thread remains a single vertical column, which is inherently mobile-friendly.

Tablet Adaptations

  • Maintains the desktop layout.

Desktop Enhancements

  • Side-by-side layout for composer actions.
  • Wider layout for better readability of long messages.

Accessibility Features

Keyboard Support
