Kanban Card (Trip)

This component provides an at-a-glance summary of a trip, designed for use in a Kanban board. It includes states for hover and dragging, and a variant for trips that are 'On Hold'.

Component Variations

Variant Name: "default"

The standard appearance for a trip card on the Kanban board. It uses a neutral color palette and relies on the status badge for state information.

Variant Name: "on_hold (highlighted)"

A visually distinct variant used for trips that are in an exception state ('On Hold'). It uses a colored left border and a light background tint based on the semantic 'warning' color to immediately draw the user's attention.
