Description

A container for a set of navigational action buttons, typically placed at the top of a form view in Odoo. Each button provides a quick link to related records, often displaying a count of those records.

Design System Alignment

Uses design system tokens for spacing, typography, colors (primary and neutrals), and border-radius. Follows standard Odoo layout patterns for form headers.

Responsive Specifications

Mobile Adaptations

  • The box uses flex-wrap, allowing buttons to stack vertically on narrow screens.
  • Maintains touch-friendly sizing for each button.

Tablet Adaptations

  • Buttons display in a horizontal row, wrapping as needed based on available space.

Desktop Enhancements

  • Hover effects provide clear visual feedback.
  • Optimized spacing for comfortable clicking with a mouse pointer.

Accessibility Features

Keyboard Support

Each button is an anchor `` tag, making it natively focusable with the Tab key and activatable with the Enter key.

Screen Reader Support

Each button has clear, descriptive text. The count and label are read together to provide full context (e.g., '3 Invoices').

Focus Management

Uses standard `:focus-visible` styling to provide a clear, high-contrast outline for keyboard navigators.

Aria Attributes

    Color Contrast

    Text and icon colors meet WCAG AA contrast ratios against the default and hover background colors.
