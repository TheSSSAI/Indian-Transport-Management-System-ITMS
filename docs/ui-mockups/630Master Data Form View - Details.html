Description

A comprehensive form view for creating, reading, and updating master data records (e.g., Vehicle, Driver, Customer). This organism combines several molecular components like the form header, smart buttons, notebook tabs, and a chatter area into a cohesive and functional interface.

Responsive Specifications

Mobile Adaptations

  • The two-column form grid collapses into a single column for better readability on narrow screens.
  • The form sheet loses its padding and rounded corners to provide an edge-to-edge experience.
  • The form header's action buttons and status bar stack vertically.

Tablet Adaptations

  • Maintains the two-column desktop layout, adapting fluidly to the available width.

Desktop Enhancements

  • A spacious two-column layout for efficient data entry and viewing.
  • Clear separation between the form sheet and the background.
  • Side-by-side arrangement of action buttons and the status bar in the header.
