Description

A communication and logging widget that displays a chronological feed of messages, system notifications, attachments, and scheduled activities related to a specific data record. It also provides a composer for users to add new notes or actions.

Design System Alignment

This component utilizes the design system's color palette for backgrounds and text, the spacing system for internal padding and margins, and the typography scale for message content and metadata. Atomic components like buttons and badges are used within it.

Responsive Specifications

Mobile Adaptations

  • The composer's action buttons stack vertically to ensure they are easily tappable and fit within narrow viewports.
  • The primary action button ('Log note') becomes full-width for easier access.
  • Padding within the feed and composer is slightly reduced to maximize content space.

Tablet Adaptations

  • The component maintains its desktop layout on tablets, adapting fluidly to the container width.

Desktop Enhancements

  • The composer features a side-by-side layout for primary action and secondary tools (attachments, activities).
  • The feed area can have a max-height with scrolling for better integration into multi-column form views.

Accessibility Features

Keyboard Support

The composer's textarea and all buttons are part of the natural tab order. Attachment links are also focusable and activatable.

Screen Reader Support

Uses semantic HTML (`

`, `
`). Author and timestamp are clearly announced. Attachment links include hidden text to specify file type and size. The composer textarea has an associated label.

Focus Management

Standard focus indicators from the design system are applied to the composer's textarea and all interactive buttons.

Aria Attributes

  • aria-label for icon-only buttons in the composer (e.g., 'Attach a file').
  • The feed container could use `aria-live='polite'` if new messages are loaded dynamically to announce them.

Color Contrast

Text and background color combinations adhere to the WCAG 2.1 AA minimum 4.5:1 contrast ratio.
