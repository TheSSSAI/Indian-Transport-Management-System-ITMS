Description

A comprehensive form for creating, editing, and viewing a single Trip record. This organism aggregates multiple molecular components like a header with statusbar, smart buttons for related records, tabbed sections for detailed information, and a chatter/log area for history and communication. It represents the central UI for managing the entire trip lifecycle.

Design System Alignment

This organism is a composition of molecular and atomic components. It utilizes the design system's grid for layout, typography for hierarchy, and color tokens for states and actions. It directly integrates the Form Header, Smart Button Box, Notebook Tabs, and Chatter molecular components.

Responsive Specifications

Mobile Adaptations

  • The main form grid collapses from two columns to a single column to ensure readability and usability on narrow screens.
  • The form header stacks vertically, with action buttons appearing above the statusbar.

Tablet Adaptations

  • The form grid may be single-column or two-column depending on the exact breakpoint and device orientation.
  • The form header may wrap if there are many action buttons.

Desktop Enhancements

  • A two-column layout is used for the main form grid to efficiently use horizontal space.
  • Full hover and focus effects are available for all interactive elements.

Accessibility Features

Keyboard Support

All interactive elements, including buttons, form fields, smart buttons, and tabs, are keyboard-focusable and activatable, following a logical tab order.

Screen Reader Support

Semantic HTML is used throughout. All form fields are associated with `

` element with `aria-current`. Tabs use appropriate `role` and `aria-*` attributes.

Focus Management

A clear, high-contrast focus ring is applied to all interactive elements on `:focus-visible`.

Aria Attributes

  • aria-label for navigation regions
  • aria-current='step' for statusbar
  • aria-selected, aria-controls for tabs

Color Contrast

All text and UI elements meet the WCAG 2.1 AA minimum contrast ratio of 4.5:1.
