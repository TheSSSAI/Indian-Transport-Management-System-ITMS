Smart Button Box

A molecular component for form-based navigation, linking to related records.

Example Usage

5 Invoices 3 Expenses Customer Ledger 2 Reports (Hover)

Component Variations

Variant Name: "with_count"

The standard smart button. It displays a prominent numerical count and a descriptive label, typically used to show the number of related records (e.g., '5 Invoices').

5 Invoices

Variant Name: "simple_link"

A simplified version of the smart button that omits the numerical count. It functions as a direct navigational link, used for actions that don't have an associated count (e.g., 'View Customer Ledger').
