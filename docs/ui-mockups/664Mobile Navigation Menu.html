Component Variations

Variant: Off-Canvas Sidebar

This is the primary and only variant of the mobile navigation menu. Its behavior is defined by its states.

States:

  • Closed (Default): The menu is hidden off-screen. The main content is interactive.
  • Open: Triggered by the hamburger icon. The menu panel slides into view from the left, and an overlay dims the main content, making it non-interactive. Focus is programmatically trapped within the panel.

The implementation above uses a CSS-only technique with a hidden checkbox to demonstrate the open/closed states without JavaScript. In a real application, the state would be managed by an OWL component's state, and JavaScript would be used for focus trapping.
