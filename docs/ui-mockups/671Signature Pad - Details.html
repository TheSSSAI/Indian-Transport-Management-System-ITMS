Description

A self-contained, mobile-first component for capturing user signatures via touch or mouse input on an HTML canvas. It provides clear visual states for empty, drawing, and signed statuses, and includes actions for clearing the signature.

Design System Alignment

The component utilizes the established design system for its interactive elements. Action buttons ('Clear', 'Save') inherit styles from the 'Large Action Button (Mobile)' atomic component, ensuring consistency in color, typography, and interaction feedback. The component's border and placeholder text use neutral color tokens, and spacing is applied using the defined spacing scale.

Responsive Specifications

Mobile Adaptations

  • The canvas and its wrapper are full-width by default within their container.
  • An `aspect-ratio` is used to maintain a usable drawing area as the width changes.
  • The component uses `touch-action: none` to prevent page scrolling while drawing.

Tablet Adaptations

  • The component scales with the container width, maintaining its aspect ratio.

Desktop Enhancements

  • The canvas supports mouse input for drawing.
  • The `cursor` changes to `crosshair` to indicate the drawing area.

Accessibility Features

Keyboard Support

Action buttons ('Clear') are keyboard focusable and can be activated with Enter/Space. The canvas itself is not keyboard-operable as it's an inherently pointer-based input.

Screen Reader Support

A `form-label` is associated with the component wrapper to describe its purpose (e.g., 'Recipient Signature'). The placeholder text provides an additional hint. The state (empty vs. signed) is communicated visually and by the presence of the 'Clear' button.

Focus Management

When the signature pad container is focused (e.g., by tabbing to the 'Clear' button), a focus ring appears, consistent with other form inputs.

Aria Attributes

  • The wrapper could have an `aria-describedby` attribute linking to help text.

Color Contrast

Placeholder text and borders meet WCAG AA contrast ratios against the background.
