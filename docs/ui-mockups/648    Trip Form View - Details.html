Responsive Specifications

Desktop Layout

  • Spacious two-column grid for the main form content.
  • Header displays action buttons and statusbar side-by-side.

Tablet Adaptations

  • The form grid may collapse to a single column depending on viewport width and orientation.
  • Maintains desktop-like spacing and touch targets.

Mobile Adaptations

  • The two-column form grid collapses into a single vertical column.
  • The form header stacks action buttons and the statusbar.
  • Padding is reduced to maximize content visibility on narrow screens.
