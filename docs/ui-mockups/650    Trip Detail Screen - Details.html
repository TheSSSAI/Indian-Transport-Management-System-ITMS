Responsive Specifications

Desktop Layout

  • A spacious two-column layout is used for the main form grid to efficiently use horizontal space.
  • All molecular components (header, smart buttons, tabs) are fully expanded and interactive.

Tablet Adaptations

  • The layout may be single-column or two-column depending on the exact width and orientation, thanks to the responsive grid.
  • Maintains desktop-like spacing and component sizes with larger touch targets.

Mobile Adaptations

  • The main two-column form grid collapses into a single vertical column for better readability on narrow screens.
  • The form header stacks vertically, with action buttons appearing above the statusbar.
  • Padding around the main form sheet is reduced to maximize content space.
