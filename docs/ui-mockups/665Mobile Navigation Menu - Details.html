Description

An off-canvas sidebar menu for mobile viewports, providing access to primary navigation links. It is triggered by a hamburger icon in the Mobile Header Bar and includes an overlay to dim the main content when open.

Design System Alignment

Uses the design system's primary and neutral color palettes, Inter typography, and standard spacing scale for padding and layout. Transitions follow the system's 'fast' timing function.

Responsive Specifications

Mobile Adaptations

  • The panel slides in from the left, covering a portion of the screen (e.g., 85% width).
  • A semi-transparent overlay covers the rest of the screen, indicating that the background content is inactive.
  • Scrolling is contained within the menu panel if the number of items exceeds the viewport height.

Tablet Adaptations

  • The behavior remains the same as mobile, as this component is specifically designed for smaller, touch-based viewports.

Desktop Enhancements

  • This component is not intended for desktop viewports, which would typically use a persistent sidebar or a top navigation bar.

Accessibility Features

Keyboard Support

The menu can be opened and closed with the keyboard. Once open, focus is trapped within the menu. Users can `Tab` through links and the close button. `Escape` key should close the menu (requires JS).

Screen Reader Support

The menu panel has `role="dialog"`, `aria-modal="true"`, and `aria-labelledby="nav-menu-title"` to announce its purpose. The overlay is hidden from screen readers with `aria-hidden="true"`. The active page link uses `aria-current="page"`.

Focus Management

Visible `:focus-visible` styles are applied to all interactive elements. JavaScript is required to trap focus within the open menu and return focus to the trigger button upon closing.

Aria Attributes

  • role="dialog"
  • aria-modal="true"
  • aria-labelledby
  • aria-hidden
  • aria-current="page"
  • aria-label for the close button

Color Contrast

Text colors and background colors for links (including active state) meet WCAG AA contrast requirements.
