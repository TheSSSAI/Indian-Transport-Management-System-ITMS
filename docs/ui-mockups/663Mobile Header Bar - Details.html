Description

A fixed navigation bar for the top of the mobile viewport. It provides context via a page title and offers primary navigation actions like 'back' or opening a menu. It is a foundational component for all screens in the mobile-first Driver Portal.

Design System Alignment

This component uses the design system's primary font family ('Inter'), color tokens (white background, gray-900 text), spacing for padding, and elevation (box-shadow) to ensure a consistent look and feel with the rest of the application.

Responsive Specifications

Mobile Adaptations

  • The component is designed exclusively for mobile viewports and is fixed to the top.
  • Title text truncates with an ellipsis if it's too long to fit the screen.
  • Actionable areas (icon buttons) have a minimum touch target size of 44x44px.

Tablet Adaptations

  • The component will stretch to the tablet width, maintaining its height and fixed position. The layout remains the same.

Desktop Enhancements

  • Not applicable. This component is specific to the mobile-first Driver Portal and is not intended for large desktop screens.
