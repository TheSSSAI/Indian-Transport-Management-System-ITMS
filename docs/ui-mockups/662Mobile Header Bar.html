My Trips

Mobile Header Bar

This molecular component provides top-level navigation and context for every screen in the mobile-first Driver Portal. It remains fixed to the top of the viewport during scroll.

Component Variations

Variant Name : "With Hamburger Menu (Dashboard/Home Screen)"

Used on the main dashboard or root screens of a section. The hamburger icon typically opens an off-canvas navigation menu. The right slot is often empty but can contain a primary action icon like 'Add'.

My Trips

Variant Name : "With Back Button (Detail Screen)"

Used on detail or secondary screens. The back arrow provides a clear navigational path to the previous screen. The title often reflects the specific item being viewed (e.g., a Trip ID).

Trip TRP-00451

Variant Name : "With Actions on Both Sides"

A more complex variant where both left and right slots are occupied. This is common for screens like a home dashboard where a menu is on the left and a primary action (like 'Add New') is on the right.

Variant Name : "With a Very Long Title"

Demonstrates how the title text gracefully truncates with an ellipsis when it's too long to fit between the left and right action slots, preventing layout issues.
