Description

A comprehensive, mobile-first view for the Driver Portal that displays all key information for a single trip. It is designed as a scrollable, single-column layout with clearly sectioned information and prominent action buttons for on-the-go driver tasks.

Design System Alignment

This organism heavily relies on the design system. It uses neutral and primary colors for its layout and typography. Spacing tokens are used for all padding and margins to create a consistent rhythm. Atomic components like 'Large Action Button (Mobile)' and 'Status Tag' are integrated and styled according to their specifications, ensuring visual consistency across the application.

Responsive Specifications

Mobile Adaptations

  • The entire view is a single, scrollable column to fit narrow viewports (360px+).
  • Sections are clearly separated by a thick border using a neutral color token, creating visual rhythm.
  • Action buttons are full-width for large, easy-to-tap targets.
  • Font sizes and spacing are optimized for readability on mobile screens.

Tablet Adaptations

  • The single-column layout is maintained, centered within the wider viewport, ensuring a consistent mobile-first experience.

Desktop Enhancements

  • This component is not designed for desktop. It is exclusively for the mobile-first Driver Portal.

Accessibility Features

Keyboard Support
